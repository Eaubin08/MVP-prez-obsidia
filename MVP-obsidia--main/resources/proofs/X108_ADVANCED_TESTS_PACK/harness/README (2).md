X-108 Test Harness Pack (pytest)

This pack contains drop-in pytest suites for:
1) Obsidia OS1 X108Gate (obsidia_os1/x108.py)
2) Trading Agent ERC-8004 gate2_x108_temporal (src/gates/gate2_x108_temporal.py)

How to use (each repo):
- Copy the corresponding folder into your repo root.
- Install pytest:  pip install -r requirements.txt  (or pip install pytest)
- Run: pytest -q

- obsidia_engine_tests/
