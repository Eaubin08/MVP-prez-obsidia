# ALGEBRE HUMAINE — TRADING AGENT

## Alphabet
A Action, M Mouvement, T Temps, S Stabilité, C Cohérence, R Risque, E Exposition, D Décision

## Opérateurs
⊕ renforce, ⊖ affaiblit, ÷ met sous tension, → entraîne, ⟂ bloque, ∧ nécessite

## Lois
L1: D = (S ⊕ C) ÷ (R ⊕ E)
L2: A → T → D
L3: Si T < τ → D ⟂
L4: Si M ⊖ S → D ⟂
L5: (S ∧ C ∧ T) > R ⇒ Entrée
L6: Si (R ⊕ E) > S ⇒ Sortie
